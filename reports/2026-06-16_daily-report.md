# 不耕起栽培 野菜 日次レポート 2026-06-16

## 今日のハイライト

- ヘアリーベッチを「刈り倒しマルチ」として活用する不耕起技術が注目。播種（秋：9〜11月）→開花期に刈り倒す→そのまま地面に敷いてマルチ→夏野菜をその上に定植、という一連の流れが完成度の高い不耕起サイクルを作る。
- ウッドチップ（剪定チップ）を厚めに敷く「Back to Eden式」不耕起マルチは灌水コストを大幅削減できるが、直播が難しいこと・発酵前のチップは病害リスクがあることが注意点。
- ネギ・ニラのコンパニオンプランツ効果に科学的な裏付けが強まっている。根圏のFlavobacterium属がフザリウム病の発生を抑制することが研究で確認されており、不耕起栽培との相乗効果が期待できる。

### 【日本】ヘアリーベッチの刈り倒しマルチ活用法 ── 不耕起野菜栽培との組み合わせ
- 出典：https://www.naro.go.jp/laboratory/carc/organic/ryokuhi-sakumotsu/hairyvetch.html / https://engeisyosinsya.com/ヘアリーベッチ / https://greenlife-lab.com/hairlybetch/
- 要点：
  - マメ科1年草または越年草。耐寒・耐雪性に優れ、日本全国で越冬可能
  - **播種時期**：暖地・中間地（関東以西）は9〜11月（秋まき主体）または3〜5月（春まき）。冷涼地は9〜10月か5〜6月中旬
  - **播種量**：3〜5 kg/10a。春まきは被覆効果を高めるため4〜5 kg/10aが目安
  - **窒素固定量**：15〜25 kg N/10a → 化学肥料の大幅削減が可能
  - **すき込み・刈り倒し適期**：開花期（草丈60〜100 cm）。茎葉が柔らかく、分解が速い
  - **刈り倒しマルチとして**：刈り倒した茎葉をそのまま地面に残すと、夏の日差し・乾燥から土を守りながら夏雑草の発芽を抑制する。秋まき→翌春刈り倒し→夏野菜定植、というサイクルが完成度が高い
  - **菌根菌増殖効果**：根圏で菌根菌が増える → 次作の野菜の根が菌根菌と共生しやすくなる（不耕起の相乗効果）
  - **アレロパシー注意**：ヘアリーベッチはシアナミドというアレロパシー物質を含むため、刈り倒し後2〜3週間は定植を控える。後作に向く野菜はナス科（トマト・ナス・ピーマン）・ウリ科（キュウリ・カボチャ）・イネ科（トウモロコシ）
  - **雑草化リスク**：種子ができる前（開花直後）に必ず処理すること。放置すると次年以降に旺盛に繁殖して制御困難になる可能性がある
- 実践のヒント：今（6月中旬）は夏野菜の最終定植期。9〜10月にヘアリーベッチを播種すれば来春の活用準備が始められる。自治体の緑肥補助事業や農業資材店で種子を入手可能（タキイ・カネコ種苗等）。

### 【国際】ウッドチップ（剪定チップ）不耕起マルチ ── Back to Eden農法の実際
- 出典：https://www.backtoedenfilm.com/no-till-gardening.html / https://www.etree.jp/content/woodreport/sustainable-0331/ / https://www.endo-kogyo.co.jp/japanese/blog/woodchips.html
- 要点：
  - Paul Gautschiが提唱。「森の床」のような環境を農地に再現する不耕起農法
  - 剪定チップを10〜15 cmの厚さで畑全面に敷き、毎年補充するだけ。耕さない、ほぼ肥料も与えない
  - 報告されているメリット：雑草の劇的な減少（2〜3年で90%減）、灌水量の大幅削減（報告では95%削減も）、土壌温度の安定化、有機物の継続補給
  - **実践上の課題（重要）**：
    - 直播が非常に難しい（厚いチップ層で種が発芽しにくい、動物が種を掘り出す）
    - 生のチップは窒素飢餓を引き起こすリスクがある
    - **対策**：半年〜1年以上発酵・熟成させたチップを使う。定植前にチップを掻き分け、コンポストを少量加えた穴を作って苗を植える
  - 日本での入手方法：多くの自治体が剪定チップ（剪定枝チップ化物）の無料または低コスト配布を実施。例：城南衛生管理組合（東京）など
  - 山の辺遊歩農園の実践例：半年以上保管して糸状菌で発酵させたチップを投入した後、病気が出なくなり野菜の味がしっかりしたと報告
- 実践のヒント：苗の定植には向くが直播は不向き。コンポストを1インチ（2〜3 cm）下敷きにして、その上にチップを戻す「コンポスト+チップ二層方式」が使いやすい。

### 【科学】ネギ・ニラのコンパニオンプランツ効果 ── 拮抗菌の科学的根拠が深まる
- 出典：https://www.ncbi.nlm.nih.gov/pmc/articles/PMC6368641/ / https://www.biorxiv.org/content/10.1101/2025.02.10.637461v1.full
- 要点：
  - Allium属（ネギ・ニラ・玉ねぎ等）の根圏では**Flavobacterium属**が優先的に増殖することが確認されている
  - Flavobacteriumは強力な拮抗菌であり、フザリウム（Fusarium oxysporum）によるキュウリ苗の萎凋病（枯れ病）を抑制することが実験で実証された
  - 東南アジアの農業研究（複数の長期研究）でも、Allium属の輪作・混植によりウリ科・ナス科・トマトのFusarium wilt（土壌病害）の発生率と重症度が有意に低下することが示されている
  - **不耕起との相乗効果**：不耕起で菌根菌ネットワークが保たれた土壌にAllium類を混植すると、微生物多様性がさらに向上し、病害抑制力が高まると考えられている
  - 2025年のbioRxiv論文では、根圏からの分泌物（バイオコントロール物質）がFusarium wiltを抑制するメカニズムを解析。ターゲット以外の微生物（非標的微生物）への影響も研究されており、土壌生態系への安全性が検証されつつある
- 実践のヒント：今すぐできること → トマト株の根元にニラを2〜3株植え込む（株間20〜30 cm）。キュウリの畝横にネギを植える。効果が出るまで1〜2シーズンかかるが、コストゼロで土壌病害リスクを下げられる。

### 【日本】不耕起栽培の学び場が全国で広がる（日本農業新聞 2025〜2026年）
- 出典：https://www.agrinews.co.jp/news/prtimes/293501
- 要点：
  - 日本農業新聞が報じた「地球温暖化を食い止める農法・不耕起栽培の学び場スタート」という記事が注目を集めている
  - 温暖化対策としての不耕起栽培（土壌炭素貯留）に焦点を当てた講座・ワークショップが全国で開始
  - 「不耕起+生き草マルチ+有機物大量投入+化学肥料不使用」の組み合わせで前年比2〜3倍増収した事例が報告されている
  - 従来「雑草と害虫が増える」と思われていた不耕起栽培が、適切に管理することで気候変動への適応農法として再評価されている
  - 自然農法センター（INFRC）が農家向けの草生栽培（不耕起×草を残す）講習を実施中
- 実践のヒント：「不耕起」への転換は一気に全畑でやると管理が難しい。まず1畝（1〜2 m²）から始めて比較実験を続けることが最もリスクが低い入り口。

### 【実践Tips】草マルチ vs マルチシート ── 梅雨時期の選び方
- 出典：検索結果の総合（https://sanchoku-prime.com/lab/articles/type-mulch / https://shop.marutoyo-japan.com/blogs/column/multi-sheet-kinds）
- 要点：
  - 有機物マルチ（草マルチ・ウッドチップ）：密閉されないため**通気性あり**。梅雨時期の過湿リスクが低い。土壌微生物のエサになり不耕起の効果を高める。資材コストが低い（刈り草なら0円）
  - 黒マルチシート：完全遮光で雑草を強力に抑制。ただし梅雨後の高温期に地温が上がりすぎてナス・ピーマンなどに高温障害が出るリスクがある
  - 白黒マルチシート：夏野菜向きで地温上昇を防ぎながら雑草も抑制。ただし土壌生態系には草マルチより不利
  - 透明マルチシート：地温を最も高く上げる（太陽熱消毒向き）。雑草は増えるため不耕起栽培には不向き
  - **不耕起栽培での推奨**：草マルチ → ウッドチップ → 白黒マルチの順。梅雨時期は草マルチで通気を確保し、梅雨明け後の高温乾燥期もそのまま保水マルチとして機能する
- 実践のヒント：草マルチは3〜5 cmが基本。梅雨時期に過湿気味なら畝の肩部分のマルチを薄くして通気を確保する。梅雨明け後は逆に厚くして保水する（厚さを季節で調整）。

## 今日の学び・まとめ

本日の調査で「不耕起栽培の年間サイクル設計」に関する情報がつながった。

> **理想的な年間サイクル（関東以西の場合）**
> - **9〜10月**：夏野菜の収穫終了後にヘアリーベッチ播種（3〜5 kg/10a）
> - **翌春3〜4月**：ヘアリーベッチ開花期に刈り倒し。そのまま地面に残してマルチとする
> - **刈り倒し2〜3週間後**（4〜5月）：夏野菜（トマト・ナス・キュウリ等）を定植
> - 定植時：株元にニラ（コンパニオンプランツ）を植え込む
> - **夏の管理**：草マルチを追加して保水。耕さない
> - **秋**：また同じサイクルを繰り返す

このサイクルを繰り返すことで：土壌有機物が増え→微生物が豊かになり→菌根菌が増え→拮抗菌が病害を抑え→水ストレス耐性が高まる、という好循環が生まれる。

初心者が今すぐできることは、トマトやキュウリの株元にニラを植えること（苗200円以下）。これだけで拮抗菌を増やし始められる。
