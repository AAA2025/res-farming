# 不耕起栽培 野菜 日次レポート 2026-06-24

## 今日のハイライト

- AM菌（アーバスキュラー菌根菌）は陸上植物の約80%と共生する「見えない相棒」。不耕起栽培ではこの菌が増えてリン酸吸収が高まり、リン酸肥料を半減させても収量がほぼ変わらないという研究結果がある。
- カニ殻（カニガラ）肥料に含まれるキチン質は「放線菌のエサ」になる。放線菌が増えるとフザリウムなど土壌病原菌の細胞壁（これもキチン質）が溶解されるという巧みな生物的防除メカニズムが働く。
- 「ローラークリンパー」はカバークロップを耕さずに倒して厚いマルチにする機械で、日本でも自作・普及が始まっている。ライ麦を秋に播いて翌春ロールで倒すだけで分厚い雑草抑制マルチが完成する。

### 【科学】アーバスキュラー菌根菌（AM菌・VA菌根菌）と不耕起栽培の相乗効果
- 出典：https://minorasu.basf.co.jp/80329 / https://pms.brc.riken.jp/ja/archives/208 / https://smartagri-jp.com/smartagri/1103
- 要点：
  - AM菌（旧称VA菌根菌）は陸上植物の約80%に共生する糸状菌。植物に光合成産物（糖）を提供してもらう代わりに、菌糸で吸収したリン酸を植物に供給する
  - **不耕起との相乗効果**：不耕起区と有機質肥料区ではAM菌の感染率が明らかに高い（エダマメの試験で確認）。耕起によって菌糸ネットワークが物理的に切断されるため、慣行耕起ではAM菌の働きが毎年リセットされてしまう
  - **「前作効果」**：トウモロコシ・大豆・ウリ科・ナス科など、AM菌と共生する宿主作物を前作にすると次作でのAM菌感染率が高まる。宿主跡地の土壌ではリン酸を半分以下に減らしても収量がほぼ変わらないという試験結果がある
  - **注意点：ネギ科は宿主外**：ネギ・ニラ・タマネギはAM菌と共生しない（感染しない）特殊な植物。コンパニオンプランツとして優秀だが、AM菌増加の観点では宿主にならない
  - **リン酸過剰は逆効果**：土壌リン酸濃度が高すぎると植物がAM菌と共生する必要を感じず、感染率が下がる。化学リン酸肥料の連続施用がAM菌を減らす一因
  - **資材の利用方法**：市販の菌根菌資材（マイコス・ハイフォスなど）は播種時に育苗土に混入するか、定植時に植穴に施用するのが効果的。少量で済み経済的
- 実践のヒント：不耕起を続けるだけでAM菌は自然に増えるが、リン酸肥料（過リン酸石灰・化成肥料のP成分）を控えると加速しやすい。最初の1〜2年はリン酸肥料を控えめにしながらコンポストを地表に置く「不耕起＋有機補給」の組み合わせが最も手軽。

### 【土壌改良】カニガラ（カニ殻肥料）のキチン質による土壌病害抑制
- 出典：https://www.noukaweb.com/crab-meal-fertilizer/ / https://gardecojapan.com/2018/03/29/post-2036/ / https://daiwa-ism.com/カニガラ粉末/
- 要点：
  - カニ殻に豊富に含まれる**キチン質（キトサン）**が土壌中の有用**放線菌のエサ**になる
  - 放線菌が増えると：①キチン分解酵素を生産 → フザリウム・リゾクトニア等の病原菌の**細胞壁（キチン質）を溶解**する、②ストレプトマイシン等の**抗生物質を生産**して糸状菌の活性をさらに抑制
  - **効果のある病害・害虫**：フザリウム萎凋病（トマト・キュウリ・ダイコン）、つる割病、萎黄病、さらにセンチュウ（外殻がキチン質）への抑制効果も
  - 化学農薬や農薬不使用の有機・不耕起農家にとって土壌病害対策として注目されている
  - **施用量の目安**：家庭菜園レベルでは100〜500 g/㎡。ただし多すぎると土壌が一時的に偏った菌相になるリスクもあるため少量を定期的に与えるほうが安全
  - **不耕起との相乗効果**：不耕起で放線菌ネットワークが壊れずに維持されているところにカニガラを投入すると、放線菌の継続的な増殖・活性化が期待できる。耕すたびに土壌微生物ネットワークをリセットする慣行農法より持続的な効果が見込める
- 実践のヒント：カニガラ粉末または粒状ペレットをコンポスト・米ぬかと一緒に畝の表面にまく。1㎡に100g（大さじ約6杯）が家庭菜園での入門量として使いやすい。ホームセンターや通販（カインズ・Amazon等）で1kg数百円程度で入手可能。

### 【農機具】ローラークリンパー ── 緑肥を「切らずに倒す」不耕起の新技術
- 出典：https://www.kk-bizen.jp/product_introduction/rollercrimper/ / https://gn.nbkbooks.com/?p=29705 / https://lib.ruralnet.or.jp/cgi-bin/ruraldetail.php?KID=202301_019
- 要点：
  - **ローラークリンパーとは**：重い金属ドラムに鈍い刃（リブ）が付いており、カバークロップ上を転圧することで茎をクリンプ（折り曲げ・つぶし）して枯死させる機械
  - 草を刈らず、根を残したまま地面に倒し、その場でマルチにする。除草剤不要で不耕起のまま厚い有機マルチが完成
  - **処理後の効果**：雑草の種が光に当たらず発芽抑制、保水、有機物補給。慣行の刈り払いよりも残渣が長持ちしてマルチとして機能する期間が長い
  - **日本での状況**：美善社（北海道）がトラクター用ローラークリンパーを製造・販売。家庭菜園〜小規模農家では手押しタイプの自作事例が増えている
  - **家庭菜園での自作事例**（現代農業2023年1月号）：管理機にユニバーサルヒッチでローラーを取り付けた自作機が紹介された。パレット等の重い板で押し倒す代替手段でも一定の効果あり
  - **最適処理タイミング**：ライ麦は出穂期〜糊熟期（6月頃）。ヘアリーベッチは開花直後。タイミングが早すぎると再生してしまう
  - **米国での課題**：小規模農家向けの二輪トラクター（BCS）用の不耕起播種機と組み合わせる製品がまだ存在しないため、家庭菜園スケールへの展開が課題（SARE 2026年研究）
- 実践のヒント：機械なしでも同様の効果を狙うなら「踏み倒し＋重石（板・レンガ）」で緑肥を水平に押し倒してから5〜7日置くと枯れ始める。その上にそのまま苗を定植（穴をあけて植える）と本物のローラークリンパーに近い効果を得られる。

### 【実践】不耕起栽培と根菜類（大根・ニンジン）── 向く・向かないの境界線
- 出典：https://agri.mynavi.jp/2024_02_24_256097/ / https://note.com/light_marten45/n/n113f2ce30d03 / https://gardenstory.jp/gardening/78233
- 要点：
  - **根菜類と不耕起の相性**は実践者の間でも意見が分かれる。大根・ニンジン・ゴボウなどの直根野菜は土の硬度に敏感で、硬い土では根が曲がったり割れたりしやすい
  - **不耕起転換初期（1〜2年目）**はまだ土が硬いケースが多く、根菜は難しい。ナス科・ウリ科・葉物から始めるのが安全
  - **不耕起3〜5年目以降**、ミミズや土壌生物が増えて土が「生物的に耕された」状態になると、根菜も問題なく育つ実践報告がある
  - **「穴あき定植法」**：不耕起の硬い土でも植え穴だけ手で掘り（15〜20cm深）、コンポストを少量入れてから種や苗を植えると育ちやすい
  - 葉物野菜（小松菜・ほうれん草・ケール）は不耕起との相性が最もよく、転換初期から問題なく栽培できる例が多い
  - **草の中への直播実験**：草を刈ってハンマーモアで粉砕し、草・土・種をサンドイッチ状にして発芽させる実験的手法も報告されている（小松菜・ラディッシュで成功例）
- 実践のヒント：不耕起を始めるなら「葉物 → ナス科・ウリ科 → 根菜類」の順番で移行するのが最もリスクが低い。大根を植えたい場所だけ最初の年に深めに（30cm）堆肥を入れて「部分的な土づくり」をしておき、翌年以降は耕さず維持するという折衷案も現実的。

## 今日の学び・まとめ

今日の調査では「土の中の見えない仲間たちとどう付き合うか」というテーマが浮かび上がった。

- **AM菌（菌根菌）**：リン酸を運ぶ「見えないパイプライン」。不耕起を続けるほど増える。化学リン酸肥料を控えることで菌との共生がより積極的になる。
- **放線菌**：カニガラを与えると増え、フザリウムなどの病原菌の細胞壁を溶かす「見えない用心棒」。
- **菌根菌（糸状菌ネットワーク）** + **放線菌** + **ミミズ** の三者が揃ったとき、不耕起栽培の効果が最大化される。

**農業初心者へのアドバイス**：
今の季節（梅雨〜夏）にすぐできることが2つある。
1. カニガラをホームセンターで購入して夏野菜の株元にひとつまみ（100g/㎡）まく → フザリウム予防と放線菌増殖が同時に始まる
2. 秋（9〜10月）に向けて「ライ麦かヘアリーベッチ」の種を用意しておく → 翌春、踏み倒して不耕起マルチとして使う

この2ステップだけで、来シーズンの不耕起栽培の土台が大きく変わる。

- ローラークリンパー自作の具体的な材料・コスト・手順（現代農業の記事詳細）
- 不耕起3〜5年目に根菜が育つようになるメカニズム（土壌硬度の変化データ）
- 草生栽培（草を残したまま野菜を育てる）の具体的な草管理：どの草を残し、どの草を抜くか
