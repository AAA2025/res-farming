# 不耕起栽培 野菜 日次レポート 2026-06-17

## 今日のハイライト

- NHK出版『やさいの時間』2026年4・5月号が「はじめての不耕起栽培」を特集。巻頭に「菜園ステージ診断表」を掲載し、雑草の種類を見るだけで自分の畑の状態が分かる方法が紹介され、国内の園芸メディアでも不耕起が主流の関心事になってきた。
- ヘアリーベッチのアレロパシー物質（シアナミド）の無害化期間が農研機構の研究で具体的に示されている：**苗の定植ならすき込みから5〜10日、直播なら5〜14日後**でOK。前回レポートの「2〜3週間待て」より早く対応できる。
- リジェネラティブ農業（環境再生型農業）が農林水産省2026年度概算要求に盛り込まれた。不耕起栽培が国の政策課題として明示されたことで、今後の普及加速が期待される。

### 【日本】NHK出版『やさいの時間』2026年4・5月号 ── 不耕起栽培の総合特集
- 出典：https://www.shuminoengei.jp/article/content/00001045.php / https://www.shuminoengei.jp/article/content/00001080.php
- 要点：
  - NHKの家庭菜園テキストが「耕すのをやめてみる？ はじめての不耕起栽培」を巻頭特集に据えた。実践編・理論編・体験記の3部構成
  - **「菜園ステージ診断表」（11ページ掲載）**：畑に生えている雑草の種類から、土壌の現状ステージを診断する手法
    - ステージ1（未熟な土）：カヤツリグサ・スベリヒユが多い。排水が悪く、有機物も少ない
    - ステージ2（中程度）：ツユクサ・メヒシバが多い。耕起栽培に向いているが有機物がまだ少ない
    - ステージ3（成熟した土・不耕起向き）：**ハコベ・カラスノエンドウ・ナズナ**が多い。土の団粒構造ができており、不耕起栽培に移行できるサイン
  - 自給菜園から大規模農家まで「ちょこっと不耕起」を試す人が急増している
  - 不耕起への第一歩：まず自分の畑の「雑草チェック」をすること
- 実践のヒント：今（6月中旬）畑を見回して、ハコベやカラスノエンドウが生えていればステージ3に近い良い土のサイン。ツユクサが多ければもう少し有機物を補給してから不耕起に移行すると安定しやすい。

### 【農研機構研究】ヘアリーベッチのシアナミド無害化期間 ── 前回疑問の回答
- 出典：https://www.naro.go.jp/publicity_report/publication/files/ryokuhi_manual08_carc20200420.pdf / https://www.cacn.jp/technology/dayori_pdf/138_mameka_fujii.pdf
- 要点：
  - ヘアリーベッチが含むアレロパシー物質は**シアナミド（cyanamide）**。農業環境技術研究所がヘアリーベッチから天然物として初めて同定した
  - **感受性が高い作物**（ホウレンソウ・ニンジンなど）への直播：すき込みから**5〜14日後**に播種すれば発芽障害リスクを低減できる
  - **苗を定植する場合**（トマト・ナス・キュウリ等）：すき込みから**5〜10日後**でOK。苗は根が発達しているためシアナミドの影響を受けにくい
  - 栽培が終わった後の土壌からは植物生育阻害物質は検出されない → 後作への影響なし
  - 刈り倒しマルチ（土にすき込まない）の場合は地上部に物質が留まるため、残渣が分解されるまでの2〜3週間を目安に定植を控えるほうが安全
- 実践のヒント：ヘアリーベッチを耕さず刈り倒して地表に残したままの場合は、苗定植まで2週間待つのが安全策。すき込んだ（土に混ぜた）場合は5〜10日に短縮できる。前回レポートの「2〜3週間」は刈り倒しマルチ限定の目安だった。

### 【草生栽培】草の管理方法 ── 刈り高・頻度・残す草の選び方
- 出典：https://agri.mynavi.jp/2024_06_16_266916/ / https://www.infrc.or.jp/activity/media/column/column-naka02/（自然農法センター）
- 要点：
  - **高刈り（こうがり）**が草生栽培の基本：根元から**10〜15cm以上の高さ**で刈る。根を残すことで土壌が固まりにくく、根圏微生物が継続的に活動できる
  - 刈り頻度：年3〜4回が目安（春・梅雨前・夏・秋）。刈りすぎると雑草が旺盛になり、刈らなさすぎると野菜の日照が妨げられる
  - **刈った草はその場に残す**（草マルチに転用）か、畝の通路部分に積む。堆肥の役割を果たし、土壌有機物を補給し続ける
  - 自然農法センター（INFRC）の報告では、不耕起×草生栽培の1年目は耕起栽培の約1/3の収穫量だったが、**労力も1/3・資材費も大幅削減**。2〜3年後から土壌が改善されて収量が回復してくる傾向がある
  - 残したい草の例：クローバー（窒素固定）、ナズナ（ステージ3の指標）、カタバミ（地被効果）
  - 抜いた方がいい草の例：ドクダミ（強いアレロパシー）、カヤツリグサ（土壌硬化のサイン）
- 実践のヒント：6月は梅雨の前後に高刈りを1回行うのがよいタイミング。草を刈ったら根元に積んで草マルチとする。刈草は乾燥させてから敷くと病害リスクを下げられる。

### 【科学】菌根菌（AM菌）と不耕起栽培 ── 効果がある野菜とない野菜の違い
- 出典：https://en.m.wikipedia.org/wiki/Arbuscular_mycorrhiza / https://smartagri.jp/mycorrhizal-fungi-agriculture-guide/ / https://minorasu.basf.co.jp/80329
- 要点：
  - **アーバスキュラー菌根菌（AM菌・VA菌根菌）**は土中に糸状のネットワーク（菌糸）を形成し、植物の根の吸水・ミネラル吸収面積を数十倍に拡大する
  - 耕すとこの菌糸ネットワークが物理的に切断される。5年間の圃場試験で、不耕起区は耕起区と比べてAM菌の根への感染率が有意に高かった（MDPI Agronomy 2016掲載論文）
  - **AM菌と共生する野菜**：トマト・ナス・ピーマン・キュウリ・カボチャ・スイカ・トウモロコシ・ニンジン・タマネギ・ニラ・ネギ など
  - **AM菌と共生しない野菜（アブラナ科＋ビーツ）**：キャベツ・ブロッコリー・カリフラワー・白菜・カブ・大根・ルッコラ・ほうれん草（※ビートも同様）
    - これらの野菜は菌根菌資材を使っても効果が出ない。有害ではないが共生関係を結ばない
  - 健康な土壌には元々AM菌の胞子が多数存在するため、**市販の菌根菌資材の添加は健全な土壌では不要**（Colorado State University Extension）。不耕起で土を守ることが最善策
  - AM菌根菌資材（キンコンバッキー等）は根の伸長が盛んな育苗期に2000倍希釈で施用すると約1ヶ月で根への共生が確認できる
- 実践のヒント：アブラナ科（キャベツ等）の後にトマト・ナスを植える輪作設計にすると、不耕起で維持されたAM菌ネットワークをトマト・ナスが最大活用できる。アブラナ科→トマト・ナス科→ウリ科の3年輪作が不耕起との相性が良い。

### 【政策】農水省2026年度予算にリジェネラティブ農業拡充が盛り込まれた
- 出典：https://agrijob.jp/contents/myagri/te / https://spaceshipearth.jp/regenerative/
- 要点：
  - **農林水産省2026年度概算要求**に「再生型農業（リジェネラティブ農業）の拡充」が柱の一つとして明記された
  - リジェネラティブ農業の定義：①不耕起栽培、②被覆作物の活用、③輪作の実践、④化学肥料・農薬の不使用 の4条件が核心
  - 現状、日本で不耕起を実践している農家は全体のわずか数%と少ない
  - 土壌炭素貯留（炭素を土の中に固定する）効果が温暖化対策として評価されており、予算措置の後押しとなっている
  - スマート農業との組み合わせ（センサーで土壌モニタリング→AI予測→不耕起管理）も今後の方向性として示されている
- 実践のヒント：国の政策として支援が始まることで、都道府県農業試験場や農業改良普及員への相談窓口が整備される見込み。お住まいの地域の農業支援センターに不耕起栽培の普及情報を問い合わせる価値が生まれてきた。

## 今日の学び・まとめ

本日の調査で「不耕起栽培の入口の作り方」と「作物ごとの菌根菌共生有無」という2つの実践的な知識が深まった。

> **今日分かった最重要ポイント**
> 1. **自分の畑が不耕起に向いているか→雑草で判断できる**：ハコベ・ナズナが生えていれば移行可能。ツユクサやカヤツリグサが多い畑はまず有機物を補給してから。
> 2. **ヘアリーベッチ後の定植タイミング**：刈り倒しマルチ→2週間待つ。土にすき込んだ→5〜10日で苗定植OK。直播は5〜14日。
> 3. **菌根菌の恩恵を受けられる野菜・受けられない野菜**：アブラナ科（キャベツ・ブロッコリー等）はAM菌と共生しない。不耕起の菌根菌効果はトマト・ナス・ウリ科・ネギ科で最大化される。
> 4. **草生栽培の草は高刈り15cm以上**。刈りすぎず、刈らなさすぎず、年3〜4回が目安。

農業初心者へのアドバイス：まず6月の今、自分の畑の雑草を10分間観察してみよう。ハコベが多ければ「不耕起への移行サイン」。スマートフォンで植物認識アプリ（PlantNet・GreenSnap等）を使えば雑草の同定も簡単。雑草は「土壌状態を知る無料センサー」だ。
