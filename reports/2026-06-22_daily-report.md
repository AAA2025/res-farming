# 不耕起栽培 野菜 日次レポート 2026-06-22

## 今日のハイライト

- 菌根菌（VA菌根菌）を増やす最大の鍵は「耕さないこと」。菌根菌ネットワークの大部分は地表から10〜15 cmの層に存在し、一度の耕起で数年分のネットワークが破壊される。不耕起を続けることが最も有効な菌根菌保護策である。
- Charles Dowdingが2026年4月に新著「Grow Together: 50 Planting Partnerships」を出版。コンパニオンプランティング×no-dig（不耕起）の組み合わせ50種を体系化。ニンジンとレタスの混植など、すぐ試せる組み合わせが多数収録。
- 「根穴（ねあな）」を活用した不耕起土づくりが注目。野菜・草の根が枯れた後に残る根穴が、次の作物の根の通り道になり、人が耕すよりも柔らかく団粒構造の発達した土ができる。

### 【科学】菌根菌（VA菌根菌）を積極的に増やす5つの方法
- 出典：https://simplifygardening.com/how-to-increase-mycorrhizal-fungi-in-soil/ / https://www.spun.earth/articles/how-to-encourage-healthy-mycorrhizal-networks-in-your-own-soil / https://www.trifectanatural.com/mycorrhizae-comprehensive-guide/
- 要点：
  - **①耕さない（最重要）**：菌根菌ネットワークの大部分は地表10〜15 cmの層に集中している。耕起はこのネットワークを物理的に切断する。不耕起を続けるだけで菌根菌は自然に回復・拡張していく
  - **②有機物を表面に追加**：堆肥・マルチ（草・ウッドチップ）を地表に積み上げることで菌根菌のエサとなる有機炭素が供給される。「掘り込まず、表面に重ねる」が原則
  - **③植物の多様性を高める**：異なる植物種は異なる菌根菌を共生させる。カバークロップ・コンパニオンプランツ・花などを混植すると菌根菌の多様性が上がる
  - **④高リン肥料・農薬を避ける**：リン酸を過剰に与えると植物が菌根菌と共生する必要性を感じなくなり、ネットワークが縮退する。合成農薬も菌根菌に直接ダメージを与える
  - **⑤菌根菌接種剤（植菌資材）の利用**：定植時に根にまぶすか、土に混ぜて使う。不耕起と組み合わせると翌年以降も効果が継続する。市販品（マイコス菌など）は種まき・定植時に活用できる
  - **注意**：ニラ・ネギ・タマネギなどネギ科（Allium属）は**菌根菌と共生しない**（非菌根植物）。ネギ科の連作は菌根菌ネットワークを育てない。ただし病害抑制（拮抗菌）の効果は別途あり（前々回レポート参照）
- 実践のヒント：定植時に苗の根を「菌根菌接種剤」にまぶすだけで接種できる。Amazonや農業資材店で「マイコライザル・ファンジャイ」「マイコス」などで検索可能。1袋500〜2000円程度。

### 【国際・最新書籍】Charles Dowding「Grow Together」2026年4月出版 ── no-dig×コンパニオンプランティング50の組み合わせ
- 出典：https://charlesdowding.co.uk/products/grow-together-50-planting-partnerships-to-boost-your-harvests / https://www.amazon.com/Grow-Together-Planting-Partnerships-Harvests/dp/B0FH18HRCS
- 要点：
  - Charles Dowding（英国のno-dig農法の第一人者）が2026年4月に発刊した最新著書
  - no-dig（不耕起）の基盤の上でのコンパニオンプランティング・混植・後作を体系化した内容
  - **収録されている組み合わせ例**：
    - レタスとニンジンの混植（レタスが地表を覆いニンジンの発芽を助ける）
    - フェンネルとほうれん草の混植（フェンネルが水分調節）
    - ニンニクとコリアンダーの混植（病害虫忌避）
    - ブリュッセルスプラウトとニンジンの混植
    - キュウリとエンドウ・イチゴの間への植え込み
  - 「succession planting（作期ずらし）」「catch-cropping（すきま栽培）」「multi-sowing（多粒まき）」も含む
  - no-digの核心哲学：「自然を模倣する。表面に有機物を積み上げ、土をできるだけ乱さない。毎年コンポストを数 cm表面に追加するだけで肥料は不要」
- 実践のヒント：Dowdingの「多粒まき（multi-sowing）」は1穴に4〜6粒まいてそのまま育て間引かない手法。日本では一般的な技法ではないが、ネギ・ニラ・バジルに応用しやすく、不耕起で狭い畝を最大活用できる。

### 【日本】「根穴（ねあな）」を活かす不耕起土づくり ── 水口文夫の実践体系
- 出典：https://www.amazon.co.jp/家庭菜園の不耕起栽培（水口文夫著） / https://gardenstory.jp/gardening/78233
- 要点：
  - 農業改良普及委員として35年の経験を持つ水口文夫氏の著書「家庭菜園の不耕起栽培 ─ 根穴と微生物を生かす」（農文協）が家庭菜園での不耕起実践のバイブル的存在
  - **根穴とは**：野菜・草の根が枯れた後に残る空洞。この穴が「生物が作る排水路・通気路」として機能し、次の作物の根の通り道になる
  - **「根穴を生かす」実践のポイント**：
    - 前作の根は引き抜かず、地際で切る。根を残して穴を作らせる
    - 根穴は目に見えないが、繰り返し不耕起を続けることで地下に複雑な根穴ネットワークが形成される
    - 団粒構造が発達すると、雨が降っても水はけがよく、乾燥しても地中に水分が保たれる
  - **「立ち枯れ作物の積み重ね」**：収穫後の残渣を捨てず、その場に倒して積み重ねる → 畑でその場腐葉土が完成する
  - 外部から持ち込む肥料・堆肥は少量で済み、土づくりコストが下がる
  - 雑草も「生きた根穴製造機」。雑草を地際で刈り、根を残すことで雑草も根穴に貢献する
- 実践のヒント：今から始めるなら「収穫後の根は絶対に引き抜かない」ルールを1つだけ決める。トマト・ナス・キュウリの根は太いので根穴が大きく、次作のダイコン・ニンジンに特に効果的。

### 【国際】不耕起菜園のための冬越しカバークロップ選び方 ── 春の処理が鍵
- 出典：https://www.gardeningknowhow.com/edible/grains/cover-crops/no-till-cover-crops.htm / https://covercropexchange.com/blog/how-to-use-cover-crops-in-your-garden / https://aigardenplanner.com/blog/post/the-ultimate-guide-to-using-cover-crops-in-no-till-gardening-263
- 要点：
  - **不耕起に向くカバークロップの条件**：越冬できる・春に刈り倒すだけでマルチになる・いずれかが豆科（窒素固定）である
  - **おすすめ品種**（英語圏の最新情報）：
    - **オーストリア冬エンドウ（Austrian winter peas）**：豆科。耐寒性が高く日本の関東以北でも越冬。窒素固定能力が高い
    - **クリムゾンクローバー（Crimson clover）**：豆科。春に美しい赤い花を咲かせる。蜜源にもなり有益昆虫を呼ぶ
    - **ソラマメ（Fava bean）**：豆科。日本でもなじみがあり、秋まき→春収穫→残渣をマルチに活用できる
    - **ウィンターライ麦（Winter rye）**：イネ科。アレロパシー効果で雑草を抑制。春に刈り倒してマルチにすると効果的（ただし日本の稲との混同注意：これは野菜畑用のライ麦で水田ではない）
  - **春の処理方法（重要）**：
    - 種ができる前（花が咲いたらすぐ、または開花直後）に根元を刈る
    - 刈った茎葉はその場に残してマルチにする（土に混ぜない → 不耕起の原則）
    - 刈り倒し後2週間待ってから夏野菜を定植する（アレロパシーが弱まる）
  - 秋に植えて冬越しさせることで、春の雑草抑制と土壌侵食防止を同時に達成できる
- 実践のヒント：日本の気候では秋まき（9〜11月）が基本。クリムゾンクローバーは市販の種も手に入りやすく（タキイ種苗など）、越冬してそのまま蜜源花壇になるため畝の周りへの混植にも向く。

## 今日の学び・まとめ

今日の調査を通じて、「不耕起栽培の効果は地下で起きている」という視点が深まった。

> **不耕起が作る地下の世界（可視化できないが重要）**
> - **根穴ネットワーク**：前作の根が分解された後に残る空洞が「自然の暗渠排水路」になる
> - **菌根菌ネットワーク**：地表10〜15 cmに張り巡らされた菌糸が植物間で養分・情報をやり取りする「地下のインターネット」
> - **団粒構造**：微生物の排泄物・死骸・根の分泌物が土粒子を結びつけてスポンジ状の構造を作る

この3つはどれも**一度耕すと壊れ、回復に数年かかる**。不耕起を続ける最大の理由がここにある。

**農業初心者へのアドバイス**：「根は引き抜かず地際で切る」を今日から実践するだけで、無意識に根穴と菌根菌ネットワークを守ることができる。コストゼロ、手間もほぼゼロの習慣変更から始めよう。
