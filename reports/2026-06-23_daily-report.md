# 不耕起栽培 野菜 日次レポート 2026-06-23

## 今日のハイライト

- ラサーニャガーデニング（段ボール重層マルチ法）は、掘り起こし不要で新しい野菜床を作る最も手軽な方法のひとつ。段ボールと草・コンポストを交互に積み、雑草を窒息させながら豊かな土を育てる。
- 茨城大学農学部の18年間研究で「カバークロップ＋不耕起」の組み合わせが土壌炭素蓄積と土壌生態系の両方を改善することが実証された。効果が出るまで複数年かかるが、継続によって耐旱性・耐過湿性が向上する。
- 2025年の大規模研究（スイス・54圃場）で、AM菌根菌（VA菌根菌）の接種成功率は土壌マイクロバイオーム指標で86%予測できると判明。低耕起・有機農業ほど在来のAM菌が増えやすく、菌根菌製剤の効果も出やすいことが確認された。

### 【日本】NHK『やさいの時間』2026年4・5月号 ── 「菜園ステージ診断」から始める不耕起栽培
- 出典：https://www.shuminoengei.jp/article/content/00001045.php / https://www.shizensaien.net/
- 要点：
  - 自然菜園講師・竹内孝功さんが、不耕起×無農薬・無化学肥料の「自然菜園」の始め方を3部構成で解説
  - **菜園ステージ診断**が第一歩：①水はけ（物理性）、②生き物の多様性（生物性）、③pH（化学性）の3項目で現在の土の状態を把握してから移行戦略を決める
  - ステージが低い（水はけが悪い・生き物が少ない・酸性が強い）場合は、いきなり不耕起に移行せず、まず有機物（堆肥・刈り草）の大量投入で地力を上げてから不耕起に切り替えると失敗が減る
  - Part3では不耕起栽培研究の第一人者・金子信博先生（福島大学）が「耕すと壊れる土壌生態系」について解説。土壌動物（ミミズ・ムカデ・ダニ類）が作る「生物性団粒」こそが不耕起で守るべき宝だと強調
  - 今号の編集チームが市民農園での実践体験を掲載：最初の1〜2シーズンは草との格闘があるが、3年目以降から畑が「自走」し始める
- 実践のヒント：今すぐ「菜園ステージ診断」をやってみる。①大雨の翌日に土の表面に水たまりが残るか（水はけ確認）、②土を5cmほど掘ってミミズが1匹以上いるか（生物性確認）、③市販の簡易pH計や試験紙でpH 6.0〜6.5かを確認（化学性確認）。この3つでステージがわかる。

### 【国際】ラサーニャガーデニング（重層型不耕起床づくり）── 段ボール×有機物で土を作る
- 出典：https://ohfb.com/book/the-no-dig-garden-a-beginners-step-by-step-guide-to-lasagna-gardening-living-soil-year-round-vegetable-harvests/ / https://warren.cce.cornell.edu/gardening-landscape/warren-county-master-gardener-articles/lasagna-gardening / https://therike.com/blogs/sustainable-living/how-to-build-a-lasagna-garden-bed
- 要点：
  - 「ラサーニャガーデニング」は掘らずに畝を作る最もシンプルな不耕起技術。別名「シートマルチング」「no-digベッド」
  - **基本の層構成（下から順）**：
    1. 段ボール（4〜6枚重ねて重ねしろを10〜15cm取る）→ 雑草・イネ科宿根草をほぼ完全に遮断
    2. 褐色（炭素）層：枯れ草・枯れ葉・剪定チップ・段ボール小片
    3. 緑（窒素）層：生草・野菜くず・コンポスト・コーヒーかす
    4. 4〜7を交互に繰り返し、最終的に厚さ45〜90cmの床を作る
    5. 一番上を完熟コンポストで仕上げる（5〜8cm）
  - 炭素：窒素比は70:30〜50:50が目安。炭素過多だと窒素飢餓が起きやすい
  - **急ぎなら苗から植えられる**：本来は6ヶ月〜1年熟成が理想だが、一番上だけコンポストを15〜20cm置いて穴を作り、そこに苗を植えれば当日から定植可能
  - **向く野菜**：レタス・バジル・ラディッシュ・ズッキーニ・トマト（苗）。根菜（ニンジン・ビーツ）は上層6〜8インチ（15〜20cm）が軟らかいことが条件
  - **注意**：段ボールに印刷のある面（光沢なし）を使い、ビニールテープ・ホチキスは取り除く。光沢のある紙・重金属インクの箱は避ける
- 実践のヒント：今（6月下旬）から段ボールを集め始めて秋の新畝づくりに備えるとちょうど良い。スーパー・電気店・ドラッグストアで無料でもらえることが多い。秋に積んでおけば来春には十分熟成した床が完成する。

### 【国際・テキサスA&M大学 2026年3月】段ボールマルチの科学的効果 ── 除草から土壌改良まで
- 出典：https://agrilifetoday.tamu.edu/2026/03/24/use-cardboard-to-stop-weeds-in-your-garden/
- 要点：
  - テキサスA&M大学農業普及サービスが2026年3月に発表した実践ガイド。段ボールを「除草剤ゼロの雑草対策」として推奨
  - 段ボールは光を完全に遮断し、下の雑草の光合成を止める。同時に微生物とミミズが段ボールを分解し、土に有機物として返す
  - **効果持続期間**：適切に施工すれば最大6ヶ月間雑草を抑制
  - **施工手順**：①段ボールを平らに広げる（大きい1枚がよい）→ ②隣の段ボールと10〜15cm重ねる → ③上に堆肥やウッドチップを7〜8cm被覆 → ④植える場所だけ段ボールを十字に切って穴を開けて苗を定植
  - **注意点**：段ボールは水を通しにくいため頭上散水では水が弾かれやすい。散水時間を1.5〜2倍に延ばすか、点滴チューブ（ドリップ）を段ボールの下に通しておくと確実
  - 多年生雑草（スギナ・ドクダミ・チガヤなど地下茎が深い種）には1シーズンでは不十分な場合あり。2〜3シーズン連続して施工すると効果が高まる
- 実践のヒント：梅雨明けの雑草ピーク前（7月上旬〜中旬）に段ボールを敷いて夏の草管理の労力を大幅削減できる。特に通路部分への施工が効果的で、畝間を段ボール＋チップで完全に覆うと素足で畑に入れるほど快適になる。

### 【科学】AM菌根菌の接種成功率と低耕起の関係 ── スイス54圃場の大規模研究（2025年）
- 出典：https://www.nature.com/articles/s41564-023-01520-w / https://pmc.ncbi.nlm.nih.gov/articles/PMC10730404/ / https://academic.oup.com/femsle/article/doi/10.1093/femsle/fnaf031/8071968
- 要点：
  - スイスの54箇所の農場で行われた大規模フィールド実験（Nature Microbiology掲載）の成果
  - AM菌根菌（アーバスキュラー菌根菌）の接種が作物生長に効くかどうかは、**土壌マイクロバイオーム指標で86%予測できる**ことが判明
  - 最も重要な予測因子は「病原性真菌の存在量」：リン・窒素などの栄養素の有無よりも、土壌に病原菌が少ないほどAM菌の接種が成功しやすい
  - 低耕起・作物多様化・有機農業の圃場では在来のAM菌が自然に豊富で、商業的な菌根菌製剤を使わなくても効果が出やすい
  - 逆に言えば、慣行耕起（毎年深く耕す）の畑に菌根菌製剤を撒いても効果が出にくい。まず低耕起・不耕起に移行することが根本的な解決策
  - 2026年の追加的な研究（FEMS Microbiology Letters）でも「接種の成功は土壌の健全性と農作物の生産性に依存する」と確認
- 実践のヒント：市販の菌根菌製剤（ハイフォス等）を使うなら、まず不耕起・有機物マルチで土を整えてから使うと費用対効果が高い。耕起畑に撒いても効果が出にくい点を理解した上で使うこと。不耕起に転換して2〜3年経った畑では「接種なし」でも自然にAM菌が増える。

### 【日本・大学研究】カバークロップ＋不耕起の18年研究 ── 茨城大学農学部
- 出典：https://agroecology.agr.ibaraki.ac.jp/カバークロップ＋不耕起栽培で温暖化の緩和と土/
- 要点：
  - 茨城大学農学部附属国際フィールド農学センター農生態システム学研究室が18年間にわたる試験栽培と土壌炭素モニタリングを実施
  - 「カバークロップ＋不耕起」の組み合わせで、①気候変動緩和（土壌への炭素貯留）と②土壌生態系改善（ミミズ・細菌・真菌の増加）が同時に達成されることを長期データで実証
  - 「不耕起では土壌動物（ミミズ・バクテリア）が土壌団粒構造を維持し、水に溶けにくい（流れにくい）土になる」という知見が得られた
  - 初期（1〜3年）は慣行耕起より収量が下がる場合があるが、継続することで確実に土壌が豊かになり最終的に安定した収量が得られる
  - カバークロップの適切な播種期間：春夏は2ヶ月以上・秋冬は4ヶ月以上の生育期間が必要
- 実践のヒント：今から秋カバークロップ（ライムギ・ヘアリーベッチ等）の準備を始める時期。9月〜10月に播種すれば4ヶ月以上の生育期間が取れ、翌春に理想的な緑肥マルチが完成する。

## 今日の学び・まとめ

今日のキーワードは「**積み重ね**」だった。

ラサーニャガーデニング（層を積み重ねる）・長期研究（年数を重ねる）・AM菌根菌（生態系を積み上げる）、すべてに共通するのは「すぐには結果が出ないが、正しい方向に向かって積み重ねれば必ず改善する」という不耕起栽培の本質的なメッセージだった。

農業初心者へのアドバイスとして今日の調査からまとめると：

> **今すぐできる3つのこと**
> 1. **段ボールを集め始める**：引越しシーズンが終わった今こそスーパーや電気店で段ボールを入手しやすい。秋の新畝づくりに向けて今から備蓄しておく。
> 2. **菜園ステージ診断をやってみる**：水はけ・ミミズ・pHの3項目を調べるだけで今の畑の課題が明確になり、どこから手をつければいいかわかる。
> 3. **今ある畝の通路に段ボール＋チップを敷く**：夏の草管理の最大の負担（通路の雑草）を段ボールマルチで一気に解決する。コストほぼゼロで梅雨明けの草管理が劇的に楽になる。
