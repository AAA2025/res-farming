# 不耕起栽培 野菜 日次レポート 2026-06-19

## 今日のハイライト

- NHK趣味の園芸の関連メディア「みんなの趣味の園芸」2026年4・5月号が不耕起栽培を巻頭特集。一般園芸ファンへの認知が急速に広まっている。
- Charles Dowdingが13年間の比較試験で証明：不耕起ベッドは耕起ベッドより**毎年平均12%収量が多い**。同量のコンポストを使用した場合の比較。
- 菌根菌（AM菌）を保護する不耕起が、野菜の収量を**20〜40%向上**させるとする研究データが蓄積されている（Cornell大学ほか複数の研究機関）。

## 見つかった記事・情報

### 【日本・トレンド】「みんなの趣味の園芸」2026年4・5月号 不耕起栽培特集
- 出典：https://www.shuminoengei.jp/article/content/00001102.php / https://www.shuminoengei.jp/article/content/00001045.php / https://www.shuminoengei.jp/article/content/00001080.php
- 要点：
  - NHK「趣味の園芸」関連Webメディアが2026年4・5月号で不耕起栽培を特集。「不耕起栽培はおもしろいし、美味しいからやめられない」という実践者の声を特集した。
  - 「耕すのをやめてみる？はじめての不耕起栽培」という入門記事では、土を耕さずに雑草・微生物・ミミズ・益虫の力を借りて「時間をかけて質の高い畑を目指す」という考え方を解説。
  - 雑草の種類で地力をチェックする方法も紹介。ハコベ・タネツケバナ・オオイヌノフグリなどが生える畑は開始OK、スギナ・イネ科の雑草が多い場合は土壌が痩せている可能性があり注意が必要。
  - 小規模な家庭菜園から大規模農家まで「少しだけ不耕起」という入り口から始める方法が広まっている。
- 実践のヒント：今の時期（6月）は夏野菜の管理期。不耕起を「今すぐ始める」なら、まず1〜2畝だけ草マルチを敷いて耕すのをやめてみる。秋の播種・定植時に向けて土の状態を観察しながら準備する好機。

### 【国際・科学】Charles Dowdingの13年比較試験：不耕起は毎年12%高収量
- 出典：https://www.saga.co.uk/magazine/gardens/no-dig-gardening-how-it-works / https://northerngardener.org/no-dig-gardening-with-charles-dowding/ / https://charlesdowding.co.uk/blogs/no-dig-resources/faqs
- 要点：
  - イギリスの農業家Charles Dowdingが、同じコンポスト量を施用した「不耕起ベッド」vs「耕起ベッド」を13年間並走させて比較。
  - **結果：不耕起が全13年で耕起を上回り、平均12%収量が多かった。** 初年度は15〜21%の差がついた。
  - 不耕起の核心は「カードボード（段ボール）＋コンポスト5cm」で開始する方法。既存の雑草・前作残渣の上に段ボールを敷き（光遮断）、その上に5cmのコンポストを置く。耕さない。
  - コンポスト層の中に直接播種・定植する。ミミズや土壌生物がコンポストを土と混合してくれる。
  - 結果として水やりが大幅に減少。合成肥料不使用でも高収量を維持できている。
  - **雑草が減る理由**：不耕起では土を掘り返さないため休眠種子が地表に出てこない。コンポスト層による光遮断効果も加わり、年を追うごとに雑草が激減する。
- 実践のヒント：今から新しい畝を作る場合、耕起ゼロで開始できる。段ボールを重ねて（2〜3枚）敷き、その上に市販コンポストまたは熟成堆肥を5〜7cm置いて、直接植え付け。費用は段ボール（0円）＋堆肥代のみ。

### 【科学】菌根菌（AMF）を保護する不耕起が収量を20〜40%向上させる
- 出典：https://www.ncbi.nlm.nih.gov/pmc/articles/PMC3036715/ / https://sustainablefoodtrust.org/news-views/more-things-in-heaven-and-earth-mycorrhizal-fungi-ploughing-no-till-and-glyphosate/ / https://www.spun.earth/articles/how-to-encourage-healthy-mycorrhizal-networks-in-your-own-soil
- 要点：
  - 菌根菌（AM菌）との共生で植物の生産性は平均**20〜40%向上**（多様な植物種での平均）。トマト・ピーマンは27〜35%の果実増加。
  - Cornell大学の研究：菌根菌は養分吸収と肥料効率を高めることで収量増加をもたらすと結論。リン・窒素・土壌有機炭素の増加も確認。
  - ニンジン・ネギへのAM菌接種の研究（PubMed）：収量増加に加え、農薬（フォキシム）の作物残留と土壌濃度が有意に低下。
  - **耕起がAM菌に与えるダメージ**：耕起（特にロータリー耕）は菌根菌の糸状ネットワーク（菌糸）を物理的に切断する。不耕起にするだけで菌糸が温存され、次の作物がすぐに菌根菌と共生できる。
  - 草生栽培（草を生やしたまま管理する不耕起）では草の根の周りにAM菌が増え、ミミズの住処が増え、土壌団粒化が進む（自然農法センターの調査）。
- 実践のヒント：AM菌を増やすには「耕さない」「除草剤を使わない」「窒素・リン肥料を過剰に与えない（高肥料はAM菌の活性を下げる）」の3点が基本。現在ある野菜の根元の土を踏み固めないことも重要。

### 【日本・実践】異常気象時代に「強い」不耕起栽培 ── 旱魃・豪雨への耐性
- 出典：https://agrijournal.jp/production/64266/ / https://mikawa.farm/shizen-saibai-kikouhendou-nouhou/ / https://gn.nbkbooks.com/?p=33821
- 要点：
  - 「極端な豪雨（50mm/h以上）」と「猛暑日（35℃以上）」の頻度が全国で年々増加している。
  - 不耕起・草生栽培が異常気象に強い理由：
    - **豪雨・浸水対策**：耕起しないため根穴や生物の通り道が残り、排水性が高い。団粒構造が発達した土は豪雨でも表土流出が少ない。
    - **旱魃・乾燥対策**：土壌有機物が豊富で保水力が高い。マルチとの組み合わせで蒸発を大幅に減少。菌根菌が植物の吸水能力を高める。
    - **猛暑対策**：有機マルチが地温の急上昇を緩和する。
  - 土壌炭素が年0.4%増加するだけで大気CO₂増加分を相殺できるという「4パーミル構想」において、不耕起は最も効率的に土壌炭素を蓄積する農法のひとつとして位置づけられている。
  - 多様な作物の混植・輪作と不耕起の組み合わせで、病害・異常気象による壊滅リスクを大幅に分散できる。
- 実践のヒント：梅雨の豪雨期（まさに今）は不耕起の効果が最も実感しやすい時期。耕起した畝では表土が流れやすく、ぬかるみも発生しやすい。不耕起畝の排水性を隣の耕起畝と比較観察すると、違いがよくわかる。

### 【実践Tips】根菜類の不耕起栽培 ── ニンジン・ダイコンのポイント
- 出典：https://saien-navi.jp/pg/blog/read/6215579/ / https://inakagurashiweb.com/archives/1173/
- 要点：
  - 不耕起では土中の「根穴」（前作の根の跡）が残るため、ニンジン・ダイコンがその穴を通ってスムーズに伸びやすい。
  - **注意点**：土中に小石や未熟な堆肥の塊が残っていると、根が二股・曲がりになる原因になる。最初の1回だけ深く耕起して石を除去し、以後は不耕起に移行する方法（「一回限り畝立て」方式）もある。
  - ニンジンとダイコンの混植で互いの成長が促進され、害虫による葉のダメージも分散できる（竹内孝功氏の自然菜園の実践）。
  - ダイコンはキャベツ・ブロッコリーの後作にすると根こぶ病リスクが増すが、不耕起で菌根菌・拮抗菌が豊富な土壌では病害抑制力が高まる。
- 実践のヒント：今（6月）から秋ダイコン（8〜9月播種）に向けて、畝を不耕起状態のまま草マルチだけ敷いて管理。播種前に草マルチをどかして、表面5cmだけを軽くほぐして播種する「ミニマム耕起」にとどめる方法が根菜には向いている。

## 今日の学び・まとめ

今日の調査で「不耕起栽培の科学的・数値的な裏付け」が一層明確になった。

**収量比較のポイントまとめ：**
| 比較項目 | 不耕起 | 耕起 |
|---|---|---|
| 年間収量（Dowding 13年平均） | 基準 | **-12%** |
| 菌根菌による収量向上 | +20〜40% | ほぼなし（菌糸が断絶） |
| 水やり頻度 | 大幅削減 | 標準 |
| 雑草（3年後） | 大幅減 | 標準 |

**初心者が今すぐできること（6月版）：**
1. 段ボール＋堆肥5cmで新しい畝を「不耕起スタート」する
2. 現在の夏野菜の株元に草マルチを3〜5cm追加して保水する
3. 梅雨期の今、耕起畝と不耕起畝の排水性・土の状態を比べて記録する

不耕起は「やめた方がいいこと（耕す）」を省くだけで、すぐに始められる。初期投資がほぼゼロな点でも、家庭菜園の入り口として最適。
