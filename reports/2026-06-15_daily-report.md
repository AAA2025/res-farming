# 不耕起栽培 野菜 日次レポート 2026-06-15

## 今日のハイライト

- **ヘアリーベッチの窒素固定力が具体的に判明**：根粒菌による窒素固定量は15〜25 kg/10a、すき込み時は33〜36 kg/10a。しかも「すき込まなくても地表に置くだけで分解される」という不耕起栽培との相性が明らかに。
- **草マルチ vs 黒マルチシート**：梅雨・高温多湿期は草マルチの「湿度調整機能」が優れ、黒マルチシートは透水性が限定的で過湿リスクがある。不耕起栽培には草マルチの方が向く理由がある。
- **ネギ・ニラの拮抗菌効果が科学的に裏付け**：根に共生する微生物が抗生物質を生成し、トマト・キュウリ・ナスの立ち枯れ病原菌を直接抑制する。コンパニオンプランツとしての効果は「偶然」ではなく菌の働き。

## 見つかった記事・情報

### 【日本・農業研究】ヘアリーベッチの不耕起活用――数字で見る窒素固定効果
- 出典：農研機構「ヘアリーベッチ緑肥マニュアル」/ 長野県諏訪地域振興局 ヘアリーベッチ導入試験 / タキイ種苗 緑肥情報
- 要点：
  - **窒素固定量**：根粒菌による窒素集積量は 15〜25 kg/10a。これは化学肥料（硫安換算）で言えば数十kg分に相当する
  - **すき込み時の窒素投入量**：地上部をすき込むと 33〜36 kg/10a の窒素が土に入る
  - **後作野菜の吸収量**：実際に後作の野菜が取り込む量は 4〜6 kg/10a（残りは後々ゆっくり分解）
  - **不耕起との相性**：ヘアリーベッチは「すき込まなくても地表に置くだけで分解・土になじむ」という特性がある。これにより不耕起でも緑肥効果が得られる
  - **雑草抑制**：つる性で地表を這って伸び、アレロパシー効果で雑草発芽を強力に抑制する
  - **自然枯れ**：6月には自然に枯れ、そのままマルチになる。管理作業がほぼ不要
  - **相性の良い前作**：キャベツ・キュウリ・トマト・ズッキーニ・バレイショなどの夏野菜の前作に最適
- 実践のヒント：秋（9〜11月）に播種し、翌年6月頃に自然枯れした後そのまま地表マルチとして活用→夏野菜を定植するだけで追肥を大幅に削減できる。冬ライ麦と混播すれば雑草抑制力がさらに高まる。

### 【比較】草マルチ vs 黒マルチシート――梅雨・夏の不耕起栽培で選ぶべきはどちら？
- 出典：産直プライム「マルチを使い分けて野菜作り！」/ AGRI JOURNAL 草マルチ解説 / 農業メディア各種
- 要点：
  - **草マルチの特性**：
    - わら・刈り草が「水分を吸ったり吐いたりする」ため、土壌湿度が自然に安定する
    - 空気層が断熱材となり、地温の急激な変化を緩和する
    - 有機物として分解され土の栄養になる（土壌微生物の活性化にも貢献）
    - **透水性が高い**：雨がそのまま土に浸透するため梅雨の過湿になりにくい
  - **黒マルチシートの特性**：
    - 太陽光を完全遮断 → 雑草抑制効果は高い
    - **透水性が限定的**：フィルム自体は水を通さないため、穴あき部分だけから水が浸透する
    - 梅雨・多雨期は畝の表面に水がたまりやすく、根腐れリスクが生じることがある
    - 地温上昇効果が高い（夏は逆効果になる場合も）
  - **不耕起栽培での使い分けの指針**：
    - 梅雨〜夏の多雨期：**草マルチが優位**（透水性・湿度調整・土壌微生物への貢献）
    - 春の地温を上げたい時期：**黒マルチが優位**（地温上昇・雑草抑制）
    - 不耕起の理念（土壌微生物を活かす）には草マルチの方が本来の趣旨に合致する
- 実践のヒント：今（6月梅雨期）は黒マルチを使っている畝でも、過湿が見られたらマルチを一部めくって通気を確保するか、来シーズンから草マルチへの切り替えを検討する。草マルチの材料は刈り草・麦わら・稲わらが手に入りやすい。

### 【日本・家庭菜園】コンパニオンプランツの王様「ネギ・ニラ」が不耕起栽培で特に効く理由
- 出典：ハイポネックスPlantia「コンパニオンプランツの代表的な品種」/ 産直プライム「ネギ科編コンパニオンプランツ」/ LOVEGREEN「コンパニオンプランツで病害虫から野菜を守ろう」
- 要点：
  - ネギ・ニラ・ニンニク・ショウガの根には特有の**共生微生物**が存在し、**抗生物質を生成**する
  - この抗生物質が土壌中の病原菌（フザリウム・リゾクトニアなど立ち枯れ病菌）を直接死滅・抑制する
  - 効果が特に確認されている組み合わせ：
    - **ネギ＋トマト・ナス・キュウリ**：立ち枯れ病・つる割れ病を抑制
    - **ニラ＋イチゴ・キュウリ**：土壌病害全般に効果
    - **ニンニク＋多くの野菜**：アブラムシ忌避効果もある
  - 不耕起栽培との相乗効果：
    - 不耕起で微生物多様性が保たれる土壌にネギ科を植えると、**拮抗菌（Pseudomonas属・Flavobacterium属）がさらに増殖**する
    - 耕してしまうとネギの根から分泌された物質と共生菌のネットワークが分断される
  - 注意：ネギ科は**同士では混植しない**（ネギ＋タマネギなど）。また豆類との相性が悪い（ネギの根が根粒菌を阻害する）
- 実践のヒント：トマト・ナス・キュウリを植える時、株元にネギやニラを1〜2本一緒に植えるだけでよい。不耕起で土をいじらないことで、この共生関係が年々強化されていく。

### 【英語・米国農業拡張機関】家庭菜園のカバークロップ完全ガイド（2026年）
- 出典：University of Vermont Extension「春カバークロップガイド」/ Iowa State University Extension「家庭菜園向けカバークロップ」/ The Old Farmer's Almanac カバークロップ
- 要点：
  - **冬ライ麦＋ヘアリーベッチの最強の組み合わせ**：
    - 冬ライ麦：他の植物の発芽を抑制（アレロパシー）→ 雑草を完全に抑え込む
    - ヘアリーベッチ：窒素固定で土を肥沃にする
    - 2種混播で「雑草対策」と「土壌肥沃化」を同時に達成
  - **不耕起の場合の刈り倒し手順**：
    1. 春の開花前（または結実前）に地際で刈り倒す（鎌・刈り払い機）
    2. 刈り倒した茎葉をそのまま地表に残す（地表マルチになる）
    3. 2〜3週間後、マルチの上から直接夏野菜を定植する（穴を開けて植える）
    4. 土は一切耕さない → 下の土壌微生物ネットワークが保たれる
  - **日本の気候での注意点**：米国の事例は乾燥気候が多い。日本の梅雨期は刈り倒し後の残渣が腐敗しやすいため、刈り倒した量が多すぎると過湿の原因になることがある。薄め（3〜5cm程度）にとどめるのが安全
- 実践のヒント（日本向け）：今年の秋（9〜11月）にヘアリーベッチ（単価安・ホームセンターで購入可）を畝に播種し、来春自然枯れしたものをそのまま地表マルチ→夏野菜定植というサイクルをまず1畝で試してみる。

## 今日の学び・まとめ

今日の調査で「不耕起栽培のピース」がさらにつながった。

**草マルチ → 微生物を生かす → コンパニオンプランツの効果が最大化** という連鎖が、科学的な裏付けと共に見えてきた。

- 草マルチは有機物を継続的に供給し、土壌微生物を活性化する
- 微生物が多様な土壌では、ネギ科のコンパニオンプランツが放出する抗生物質物質と共生菌の相乗効果が高まる
- 不耕起（土をいじらない）ことで、この生態系ネットワークが年々安定・強化される

**今（梅雨期・2026年6月）にすぐできること**：
1. **草マルチを試す**：刈り草を畝の上に5cm程度敷く（黒マルチより通気・透水性が良く梅雨に向く）
2. **ネギまたはニラを混植**：夏野菜の株元に1〜2本差し込むだけ。抜かない・耕さないことで効果が続く
3. **秋のヘアリーベッチ播種を計画**：9〜11月に播種すれば来年の夏野菜前作として機能する
