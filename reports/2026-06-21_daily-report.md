# 不耕起栽培 野菜 日次レポート 2026-06-21

## 今日のハイライト

- 草生栽培での**草の管理方法が具体化**：草丈40cmを目安に刈り、刈り後10〜15cmを残す。大鎌が刈り払い機より優れている理由とは？
- 菌根菌（VA菌根菌）は**アブラナ科・アカザ科の野菜とは共生しない**。不耕起栽培の恩恵が得られない野菜があることを輪作設計に活かす必要がある。
- **ラサーニャガーデニング**（段ボール＋層別堆肥法）が新しい不耕起畝づくり手法として注目。耕さずに新しい畝が作れ、秋に仕込めば翌春に準備完了。
- 前回レポートで「次回調べたい」と挙げた**ウッドチップ無料配布の全国自治体例**を今回まとめた。

## 見つかった記事・情報

### 【日本】草生栽培における草の管理実践 ── 刈り高さ・頻度・使う道具
- 出典：https://www.infrc.or.jp/activity/media/column/column-naka02/ （自然農法センター）/ https://agrijournal.jp/production/64266/
- 要点：
  - **刈り高さの基準**：草丈が40cm前後になったタイミングで刈る。刈り後の草丈が10〜15cmになるよう、茎のしっかりした箇所を狙う
  - **頻度の目安**：春〜初夏（生育が速い時期）は約2週間に1回。涼しくなる秋以降は1ヶ月に1回程度
  - **道具の選び方が重要**：
    - 刈り払い機（チェーンソー型）→ 地際まで刈りすぎて土が乾燥しやすくなり、一年生雑草（メヒシバ・エノコログサ）が増加しやすい
    - 大鎌（手刈り）→ 地際から10〜15cmの適切な高さで刈れ、再生が早く、多年生草が育ちやすいため長期的に雑草管理が楽になる
  - **刈った草の扱い**：必ずその場の敷草（マルチ）として残す。持ち出さない。炭素・窒素の物質循環をその土壌内で行わせることが目的
  - **施肥は無肥料が推奨**：肥料を与えると草生帯の植物が旺盛になりすぎ、害虫・草刈り回数増加・野菜の生長阻害につながる
  - 草が茂みになるほど放任するのはNG：害虫・野鼠の巣になり、景観も悪化
- 実践のヒント：梅雨に入ると草の伸びが急速に速くなる。今（6月下旬）が大鎌を使い始めるタイミング。最初は草丈が気になるかもしれないが、10〜15cmで止まった草は土壌を守るグラウンドカバーとして機能する。

### 【科学】菌根菌（VA菌根菌）と共生しない野菜 ── アブラナ科・アカザ科の注意点
- 出典：https://jspp.org/hiroba/q_and_a/detail.html?id=1884 （日本植物生理学会）/ https://ecologia.100nen-kankyo.jp/column/single145.html
- 要点：
  - アーバスキュラー菌根菌（AM菌根菌）は**約80%の陸上植物と共生**するが、一部の植物とは共生しない
  - **AM菌根菌と共生しない主な野菜（不耕起の恩恵が限定的）**：
    - アブラナ科：キャベツ、ダイコン、ハクサイ、コマツナ、ブロッコリー、カブ
    - アカザ科：ホウレンソウ
  - **共生する主な野菜（不耕起で最大の恩恵）**：
    - ナス科（トマト、ナス、ピーマン）、ウリ科（キュウリ、カボチャ）、セリ科（ニンジン）、マメ科（エダマメ、インゲン）
  - **重要な発見**：アブラナ科を栽培する場合でも、前作に宿主植物（ナス科・ウリ科など）を栽培しておくと、土壌全体の菌根菌密度が維持され、アブラナ科の環境ストレス耐性が高まるとされている
  - 不耕起で菌根菌ネットワークを壊さないことで、宿主作物の生育が飛躍的に向上する
- 実践のヒント：輪作計画を「菌根菌と共生する野菜」→「アブラナ科」の順で組むと効果的。例：春トマト（菌根菌宿主）→ 秋ダイコン（非宿主）という順番ならダイコン前に菌根菌を増やせる。アブラナ科からアブラナ科への直接連作は菌根菌密度が下がり続けるため注意。

### 【国際・実践】ラサーニャガーデニング ── 段ボール＋層別堆肥で不耕起ベッドを作る
- 出典：https://extension.oregonstate.edu/catalog/em-9559-sheet-mulching-lasagna-composting-cardboard （オレゴン州立大）/ https://agrilifetoday.tamu.edu/2026/03/24/use-cardboard-to-stop-weeds-in-your-garden/ （テキサスA&M大 2026年3月）/ https://warren.cce.cornell.edu/gardening-landscape/warren-county-master-gardener-articles/lasagna-gardening （コーネル大）
- 要点：
  - 「ラサーニャガーデニング」＝シートマルチ＋層別堆肥法。「ラサーニャ」のように材料を重ねることから名前がついた
  - **基本の作り方（5ステップ）**：
    1. 地面に水を十分に撒き、雑草を踏み倒す（抜かない）
    2. 段ボール（プレーンな茶色の段ボール、糊・テープは除去）または新聞紙を敷く。10cm以上の重ね代をつけて目地をふさぐ。水で湿らせてから敷くと固定しやすい
    3. 「茶色（炭素）資材」を5cm積む：落ち葉・藁・おがくずなど
    4. 「緑（窒素）資材」を5cm積む：草刈りくず・生ゴミ・腐葉土・米ぬかなど
    5. 3〜4を交互に繰り返す。最低30〜50cmの厚さが目安
  - **仕込み時期**：秋（9〜10月）に仕込むと冬の間に分解が進み、翌春には畝として使える。春先から仕込む場合は定植まで最低3ヶ月待つ
  - **段ボールの役割**：雑草を遮光して枯らし（光遮断）、ミミズを呼び込み、分解されて土になる。除草剤ゼロで既存の雑草を根絶できる
  - **2026年のテキサスA&M大学の推奨記事**では、段ボールを雑草対策の「安全・持続可能な第一選択肢」として正式に推奨。ただし印刷・コーティングされた段ボールは使用不可
  - 分解完了まで数ヶ月〜1年かかるため、急いで使いたい場合は直接定植穴を掘って植える方法（穴あけ定植）も可能
- 実践のヒント：家電の大型段ボールや引越し用の段ボールが最適。今すぐ畳んで保管しておけば秋の仕込みに使える。米ぬか（緑資材として）は近所の精米機から無料入手できることも多い。

### 【日本・実践情報】ウッドチップ・剪定チップの自治体無料配布 ── 全国事例まとめ
- 出典：https://kahachiya.com/apps/note/barkchip-muryou/ / 各自治体ホームページ（原村・富士見町・諏訪市・奈良市・宇都宮市・山梨県・城南衛生管理組合）
- 要点：
  - 全国の多くの自治体で、街路樹・公園・庭木の剪定枝をチップ化したものを**無料または低価格で配布**している
  - **配布自治体の例（2025〜2026年確認分）**：

    | 地域 | 配布元 | 備考 |
    |---|---|---|
    | 東京（城南地区） | 城南衛生管理組合 | 先着順・事前申込制 |
    | 長野県 原村 | 原村役場 | 電話または電子申請で申込 |
    | 長野県 富士見町 | 富士見町 | 剪定枝の受け入れと同時にチップを配布 |
    | 長野県 諏訪市 | 諏訪市 | 市民に無料配布 |
    | 奈良市 | 奈良市 | 家庭菜園利用可と明記 |
    | 宇都宮市 | 民間資源化事業者経由 | 市の案内で無料入手可 |
    | 山梨県 | 思川開発建設所（ダム管理） | ダムの流木由来チップ |

  - 配布品は「使用前に半年〜1年熟成させる」のが推奨（生のチップは窒素飢餓リスクあり）
  - 品質にばらつきがあり、プラスチックや金属が混入している場合もあるので受け取り後の目視チェックが必要
  - **入手方法**：各自治体の「剪定チップ 配布」で検索、または市区町村のごみ・緑化担当課に問い合わせる
- 実践のヒント：「ウッドチップ 配布 ＋（住んでいる市区町村名）」で検索すれば地元の配布情報が見つかることが多い。ダム管理所のチップは量が多く、農地規模での活用に向いている。

## 今日の学び・まとめ

今日の調査で「不耕起栽培の野菜別・段階別の使い分け」についての理解が深まった。

**まとめると：**

1. **菌根菌の恩恵を最大化できる野菜**（不耕起の核心）：トマト、ナス、キュウリ、ニンジン、エダマメなど → これらを中心に不耕起で作る
2. **菌根菌が機能しない野菜**（アブラナ科・ホウレンソウ）→ 前作に宿主植物を置けば土壌菌根菌密度を維持しながら栽培できる
3. **草生栽培の草管理**：「大鎌で高め（10〜15cm残し）」に刈ることが土壌保護と雑草管理の両立に最適
4. **新しい畝の作り方**：ラサーニャガーデニングで耕さずに新畝が作れる。秋仕込みが理想

農業初心者へのアドバイス：「不耕起にしたのに虫が増えた」「雑草が多い」と感じた場合、草の刈り方（高さ・頻度）を見直すと改善することが多い。刈り過ぎ（地際まで刈る）は土を乾燥させて一年生雑草を増やす逆効果になる。大鎌を一本持っておくことが不耕起草生栽培への近道。
