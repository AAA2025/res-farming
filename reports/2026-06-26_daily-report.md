## 今日のハイライト

- **雑草は土壌の「診断書」**：生えている雑草の種類を見るだけで、土壌の水はけ・pH・肥沃度が分かる。不耕起転換前の「菜園ステージ診断」として活用できる実践的ツール。
- **コンパニオンプランツ×不耕起の相乗効果**：不耕起で土壌微生物ネットワークが維持された畑にコンパニオンプランツを組み合わせると、農薬・化学肥料ゼロでも夏野菜が安定して育つ体系が作れる。
- **ナメクジ問題の正体と対策**：有機マルチを厚くすると確かにナメクジが増えやすい。ただし「敷き藁はNG・燐酸第二鉄ベイトはOK（有機JAS対応）」という明確なルールがある。

### 【土壌診断】雑草を読む ── 生えている草が教える土壌の状態
- 出典：[生えている雑草で土壌診断！ 雑草が教えてくれる4つのこと - agri mynavi（2025年7月）](https://agri.mynavi.jp/2025_07_18_338543/) / [What Weeds Say About Your Soil - Old Farmer's Almanac](https://www.almanac.com/what-weeds-tell-you-about-your-soil) / [雑草は畑の状態を表す指標になる - かく一](https://www.kaku-ichi.co.jp/media/crop/earth-building/utilization-of-weeds)
- 要点：
  - 雑草は自分が育てる条件（土壌のpH・水分・肥沃度・硬度）が整った場所にしか生えない。裏返せば「どの雑草が多いか」で畑の現状が分かる
  - 不耕起栽培の第一歩は「菜園ステージ診断」から。水はけ・生物多様性・pHを雑草から読み取ることで、最初に何をすべきかが分かる

  **日本でよく見られる指標雑草リスト**：

  | 雑草（日本名） | 土壌の状態のサイン |
  |---------------|-------------------|
  | **ドクダミ・チドメグサ・カヤツリグサ** | 水はけが悪い・過湿 |
  | **スギナ（ホーステール）** | 水はけが悪い・酸性・pH低め |
  | **オオバコ** | 土が締まって硬い・酸性・低肥沃 |
  | **タンポポ・イタドリ** | 土が締まって硬い（特に粘土質） |
  | **スベリヒユ（パースレイン）** | 肥沃・リン酸過多 |
  | **ヒユ（ピッグウィード）** | 肥沃・窒素が豊富 |
  | **ギシギシ（ドック）・ハコベ** | 過湿または豊かな土（種類による） |
  | **スイバ・イラクサ** | 酸性土壌 |
  | **メヒシバ・スベリヒユ・オオバコ** | 水はけが良い土壌 |

  - 重要な注意：雑草はあくまで「指標」であり、決定的な診断ではない。特に多年草雑草が指標としての信頼度が高く、一年草は他の要因で生える場合もある。土壌酸度計や簡易pH測定キットと組み合わせるのが理想
  - **不耕起3年目以降の変化**：不耕起を続けると土が柔らかくなり、最初は「締まった土指標」だったオオバコが減ってスベリヒユ（肥沃指標）が増えてくる。これが不耕起が土を変えている証拠

- 実践のヒント：今の畑に何の草が多いかを記録してみよう。スギナが多い→酸性対策で苦土石灰（不耕起では表面散布のみ）を施用。ドクダミが多い→高畝で水はけを改善するか、通路に溝を入れる。タンポポが多い→ミミズを増やす有機物補給で自然に改善する。

### 【害虫管理】不耕起・有機マルチ下でのナメクジ・ダンゴムシ対策
- 出典：[畑のナメクジ対策・無農薬で防ぐ方法 - カジトラ](https://kajitora.com/namekuji-taisaku/) / [ナメクジを駆除・防除する農薬と農薬以外の方法 - 農家web](https://www.noukaweb.com/slug-pesticide/) / [9 Organic Ways to Get Rid of Slugs - Homestead and Chill](https://homesteadandchill.com/organic-slug-snail-control/) / [Organic Pest Control 2026 Blueprint - GardenersYards](https://gardenersyards.com/organic-pest-control-blueprint-2026/)
- 要点：
  - **なぜ不耕起でナメクジが増えるのか**：有機マルチが湿った環境を保つことで、ナメクジが好む「涼しくて暗い・湿った場所」が年中確保されてしまうため。特に梅雨時期（6〜7月）が要注意
  - **マルチの種類で大きく差が出る**：「敷き藁」はナメクジの産卵場所・越冬場所になるため被害が拡大しやすい。対してウッドチップや刈り草（生草）は乾燥が速く、ナメクジが嫌う。可能なら梅雨時期だけ畝周辺のマルチを薄くするのが有効
  - **有効な対策（有機・無農薬）一覧**：

  | 方法 | 効果 | 備考 |
  |------|------|------|
  | **手で取る（夜間巡回）** | ★★★★★（87%成功率） | 夜〜早朝、雨の日翌朝が狙い目 |
  | **燐酸第二鉄ベイト（スラゴ等）** | ★★★★★（86%成功率） | 有機JAS対応。天然成分で土に還元される |
  | **珪藻土の散布** | ★★★★（84%成功率） | 体表の水分を奪う物理的防除。雨で流れるので再散布必要 |
  | **ビールトラップ** | ★★★★（80%成功率） | 容器にビールを入れて埋める。安価で手軽 |
  | **銅テープ・銅板** | ★★★ | ナメクジの粘液と反応して微電流が発生し忌避する |
  | **木酢液（200〜500倍希釈）** | ★★ | ダンゴムシにも有効。定期的な散布が必要 |

  - **ダンゴムシについて**：ダンゴムシは基本的には枯れ葉・有機物を食べるため益虫的な役割もある。問題になるのは苗の根元や新芽を食べる「食害」のとき。木酢液または食酢（5%溶液）のスプレーで忌避効果あり
  - **長期的な対策（生態系アプローチ）**：ナメクジの天敵（カエル・トカゲ・コウモリ・ムカデ・アシナガバチ）が住める環境を維持することが最も持続的な対策。不耕起畑で生物多様性が高まると自然にナメクジ密度が下がる報告あり

- 実践のヒント：今すぐできる最安・最効率はビールトラップ（空き容器にビールを入れて地面に埋める）×夜間手取りの組み合わせ。燐酸第二鉄ベイトは雨の日でも流れにくく長持ちするため、梅雨時期の常備品として揃えておくと安心。

### 【草生栽培】夏の草管理の実際 ── いつ刈る、何を残す
- 出典：[リジェネラティブ農法・不耕起栽培のやり方 - miufarm](https://miufarm.com/no-till-farming/) / [土壌を豊かにする不耕起・草生栽培 日本での現状 - AGRI JOURNAL](https://agrijournal.jp/production/64266/) / [無施肥・不耕起の草生栽培 - 自然農法センター](https://www.infrc.or.jp/activity/media/column/column-naka02/)
- 要点：
  - **基本ルール「高さ40cmで刈る」**：草を生やしっぱなしにしていると野菜が光不足になる。草の高さが40cm前後になったら刈り、その場に敷いてマルチとする
  - **刈り頻度の目安**：
    - 春〜初夏（成長期）: 2週間おきが目安
    - 夏（梅雨明け後）: 月1〜2回
    - 秋（涼しくなった後）: 月1回
  - **残す草・刈る草の判断基準**：

  | 残す草（益草） | 刈る・除く草 |
  |--------------|------------|
  | クローバー類（地面を覆い低く広がる） | ツル性の草（ヒルガオ・クズ・カラスウリ） |
  | ハコベ（天敵昆虫の生息地） | 大型化する草（ヨモギ・セイタカアワダチソウ） |
  | カタバミ（光を通す薄い地被） | 種を大量に飛ばす草（イヌビエ・メヒシバ） |
  | ヨモギの若芽（通路のみ、小さいうち） | 野菜を覆う高さになったもの全般 |

  - **「草を根元で切らない」という逆転の発想**：草を地際ではなく茎の中間で刈ると、根が地中に残って根穴が維持される。また再生した草が低く密なカーペットになり、後の草管理が楽になる
  - **刈り草の活用方法**：刈った草は捨てずにその場に敷く。刈り草の積み重なりが増えるほど：①雑草種子が発芽しにくくなる、②地温が下がる、③水分が保たれる、④ミミズが増えて土が豊かになる
  - **福島での実験事例**（あだたら食農School farm）：草を刈る前に野菜の種を草の上にばら播きし、その後草を刈り倒すと、刈り草マルチの中で発芽した野菜が育つ「草の中への直播」が成功した事例がある（小松菜・ラディッシュ）
  - **無施肥草生栽培の注意点**：肥料を与えると草が過繁茂して、逆に野菜が埋もれてしまう。草生栽培は基本「無施肥」か「少量の有機物地表施用」が原則

- 実践のヒント：梅雨明け直後（7月上旬）が草の最盛期。この時期に1回しっかり刈り、刈り草を畝に10cm以上積んでおくと、8月の猛暑をマルチが守ってくれる。草刈り機（ハンマーモア）がなければ鎌で刈った草を「その場に横置き」するだけで十分。

### 【科学】不耕起栽培の雑草抑制効果 ── 3年で劇的変化
- 出典：[9 No-Till Gardening Methods That Naturally Suppress Weeds - FarmstandApp](https://www.farmstandapp.com/6569/exploring-the-benefits-of-no-till-gardening-for-weed-management/) / [No-Till Gardening: Boost Soil Health - Garden Design](https://www.gardendesign.com/how-to/no-till-garden.html) / [No-Till Gardening builds healthier soil - OSU Extension](https://extension.oregonstate.edu/news/no-till-gardening-builds-healthier-soil-home)
- 要点：
  - **不耕起3年後の驚くべき数字**：2015年の研究によると、不耕起3年目の畑では雑草種子の発芽率がわずか **4%** に低下。対して耕起区では **33%** の発芽率が続いていた
  - **なぜ不耕起で雑草が減るか**：土を耕すたびに深層の雑草種子が表面に出てきて発芽する。不耕起ではこの「雑草種子バンクの撹乱」が起きないため、年々雑草の発芽が減る
  - **水の浸透率が50%向上**：ミシガン州立大学（MSU Extension）の研究で、長期不耕起農場は耕起農場と比較して土壌への水浸透率が最大50%高いことが確認された。この水はけの良さが豪雨・台風時の冠水リスクを大幅に低減する
  - **土壌バクテリア・菌類が増加**：サウスダコタ州立大学の研究で、長期不耕起農場は耕起農場より土壌バクテリアと菌類の量が「実質的に多かった」と報告された
  - **重要な含意**：不耕起は「最初の1〜2年が最も大変」で、3年以降は雑草管理が自動的に楽になっていく。初期の苦労を乗り越えるほど長期的リターンが大きい

- 実践のヒント：「不耕起にしたら雑草が大変になった」と感じるのは転換初年度によくある現象。3年目まで続けると、同じ時間の草管理で劇的に楽になる。記録（写真・重量比較）をつけておくと変化が実感しやすい。

## 今日の学び・まとめ

今日の調査では **「雑草と不耕起栽培の深い関係」** と **「有機マルチの副作用としてのナメクジ問題」** という二つのテーマを深掘りできた。

**雑草は敵ではなく、情報源である。**

不耕起を始める前に今の畑の雑草を観察することで、
- 水はけが悪いのか（ドクダミ・スギナが多い）→ 高畝を先に作る
- 土が硬いのか（オオバコ・タンポポが多い）→ 段ボール＋堆肥式で表面から土を育てる
- すでに土が肥えているのか（スベリヒユ・ヒユが多い）→ 追肥不要で始められる

という「最初の一手」が変わってくる。

**ナメクジ問題はマルチ選びで半分解決できる。**

敷き藁のような湿りやすいマルチよりも、ウッドチップや刈り草（乾燥しやすいもの）を選ぶことが根本的な予防策。梅雨時期だけ燐酸第二鉄ベイトを常備しておくのが実践的な対策。

農業愛好家へのアドバイス：今日から「雑草日記」をつけてみよう。畑に生えている草の名前を調べて、それが示す土壌状態と照合すると、自分の畑が「何を必要としているか」が見えてくる。スマホの植物識別アプリ（PictureThisなど）を使えば草の同定も簡単。
